# GIF CLI Parser

## Описание

Консольный парсер GIF-файлов на Python.  
Позволяет читать структуру GIF, выводить информацию о заголовках,
декодировать изображения с помощью LZW и отображать изображения
и анимации прямо в терминале без использования графических библиотек.

## Возможности

- CLI-интерфейс
- Парсинг заголовков GIF (Signature, Version, Size, Color Table)
- Декодирование изображений (LZW)
- Поддержка анимаций и задержек кадров
- Прозрачность
- Попиксельная отрисовка в терминале (ANSI TrueColor)
- Масштабирование под размер терминала
- Корректная обработка маленьких и больших изображений

---

## Использование

Запуск парсера:

```bash
python -m parser <file.gif>
```

Просмотр справки:

```bash
python -m parser -h
```

Вывод заголовков:

```bash
python -m parser example.gif --headers
```

Показ изображения или анимации:

```bash
python -m parser example.gif --show
```
